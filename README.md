# Simple Calculator

## Description
A simple calculator application created using HTML and CSS. This project showcases the fundamental principles of creating a functional
calculator interface without relying on JavaScript.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Clean and user-friendly interface.
- Fully responsive design for better usability on different devices.
